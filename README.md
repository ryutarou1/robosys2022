# Plus プログラム

## 概要
このプログラムは、標準入力から数値を読み取り、それらの合計を計算して標準出力に出力する Python スクリプトです。

## 実行方法
-----
1.  GitHub からリポジトリをクローンします

　git clone https://github.com/ryutarou1/robosys2022.git

2. 標準入力から数値データを入力します。

3. ./plusでスクリプトを実行します。
----
## 実行例
----
$ seq 3 | ./plus   # 正常な入力

6

$ echo $?

0

$ echo あ | ./plus  # ひらがなを入力してエラーを起こさせる

'あ' にアクセスできません: そのようなファイルやディレクトリはありません

$ echo $?

1
----

Python 3
